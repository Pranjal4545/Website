Software - Introduction

Stand-alone hardware cannot perform tasks without software. Software helps mobilize hardware and resources by providing a set of instructions, known as programs. The term software refers to the collection of related programs and associated documentation. Hardware and software must work together to produce useful output.

Types of Software

1) System Software

System software controls the operation of the computer system. It includes:

1.1) System Management
  • Operating System: Controls overall operation; examples include Windows and Linux.
  • Utility Software: Performs specific tasks for maintenance; examples include Norton Utilities.
  • Device Driver: Ensures hardware devices function correctly.
1.2) System Development
  • Programming Language: Set of keywords and symbols for writing programs.
  • Language Translator: Converts programs from one language to another; includes compilers and interpreters.

2) Application Software

Application software is designed to perform specific tasks for users. It includes:

2.1) Packaged Software:

Generalized programs for broad use; examples include MS Office.

2.2) Customized Software:

Tailored software for specific user needs; examples include payroll systems.

Operating System - Introduction

An operating system (OS) is a collection of programs that manage computer resources such as CPU, memory, and I/O devices. It provides a platform for application programs to run and facilitates user interaction with the computer.

Functions / Features / Advantages of Operating System (OS)

  • Input-Output (I/O) Management: Manages communication with peripheral devices.
  • User Interface (UI): Provides a user-friendly environment; includes CUI and GUI.
  • Security: Protects against threats and unauthorized access.
  • Process Management: Allocates CPU time to processes.
  • Memory Management: Manages memory allocation for programs.
  • Data Management: Organizes data into files.
  • Command Interpreter: Translates user commands into executable instructions.
  • Dead-Lock Prevention: Prevents deadlocks in resource allocation.
  • Time Sharing: Allows multiple users to share computing resources.
  • Virtual Memory: Uses secondary storage to extend available memory.

Types of Operating System (OS)

A) On the Basis of Processing

  • Multi-programming OS
  • Multi-tasking OS
  • Multi-processing OS
  • Time-sharing OS
  • Batch processing OS
  • Multi-threading OS
  • Real-time OS
  • Online processing OS

B) On the Basis of User Interface (UI)

  • Character User Interface (CUI)
  • Graphical User Interface (GUI)

C) On the Basis of Mode of User

  • Single User
  • Multi User

Some Important Operating Systems in Use

  • Open Source Operating System: Examples include Linux and Unix.
  • UNIX: A powerful, multi-user OS.
  • Linux: An open-source OS based on UNIX.
  • MAC OS: A GUI-based OS for Apple computers.

Terminologies Related to OS

  • Spooling: Balances the speed difference between CPU and I/O devices.
  • Process: A program in execution.
  • Thread: A smaller sub-process of a program.
  • Scheduling: Determines which process runs first.
  • Buffering: Temporary storage during I/O operations.
  • Virtual Memory: Technique for using secondary storage as additional RAM.
