MS-Word (Word Processor)

MS-Word is a text editing software developed by Microsoft Corporation. It enables users to create, edit, and print documents.

Features of MS-Word

  • Fast Operation: Faster than mechanical processes.
  • Editing Text Feature: Allows easy modification of text.
  • Permanence Storage: Documents can be stored electronically.
  • Graphics: Insert images and graphs.
  • Mail Merge: Produce multiple letters and labels easily.
  • Spelling and Grammar Checker: Notifies users of errors in documents.

MS-Excel (Spreadsheet)

MS-Excel is a spreadsheet software that allows users to store, organize, and analyze data.

Features of MS-Excel

  • Multiple Worksheets: Work with several sheets at the same time.
  • Charts and Graphs: Convert data into visual formats.
  • Automatic Calculations: Use formulas to perform calculations.

Common Terminologies

Term Description
Worksheet A collection of rows and columns for data.
Workbook File containing multiple worksheets.
Cell Intersection of a row and column.

MS-PowerPoint (Presentation)

PowerPoint is presentation software used to create dynamic presentations with slides.

Features of MS-PowerPoint

  • Formatting: Make documents attractive.
  • Slide Transitions: Add effects between slides.
  • Charts and Graphs: Include visuals to convey information.
