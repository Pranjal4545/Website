Introduction

Computers do not work by themselves; they need commands. These commands are written in programming languages.

Qualities of Good Programs

  • Easily understood
  • Correct and error-free
  • Reliable
  • User-friendly interface
  • Portable and flexible

Types of Programming Languages

Low Level Language

  • Machine Level Language: Consists of binary code.
  • Assembly Language: Uses mnemonics to represent instructions.

High Level Language

  • Procedural Oriented Language: Focuses on procedures; examples include C and FORTRAN.
  • Object Oriented Language: Focuses on data; examples include Java and C#.
  • Natural Language: Uses simple sentences for programming; currently in development.

Language Translators

  • Assembler: Converts assembly language to machine code.
  • Compiler: Converts high-level language to machine code.
  • Interpreter: Translates high-level language line by line.

Common Terminologies in Programming

Term Description
Identifier Name for entities in a program.
Keyword Reserved words in a programming language.
Data Type Specifies the type of data used in a program.
