Introduction to the Internet

The Internet is the global network of networks, allowing computers worldwide to connect and communicate.

Components Required for Internet Connections

  • A TCP/IP enabled computer with a web browser.
  • An account with an ISP (Internet Service Provider).
  • A telephone line.
  • A MODEM to connect to the telephone line and computer.

Internet Services and Applications

  • World Wide Web (WWW): A collection of interlinked documents.
  • Web Browser: Software to access web pages.
  • FTP: File Transfer Protocol for moving files.
  • E-mail: Sending and receiving messages electronically.
  • Video Conferencing: Real-time communication with video.

Internet Terminologies

Term Description
ISP Provider of Internet access.
URL Unique address for a web resource.
Search Engine Software that searches for information online.
